Various projects that I've worked on in order from most recent to oldest.

Unnamed Chat Program (Google CodeU Summer 2017)

A provided basic framework for a Java chat application that was improved upon over the course of a few months.

Improvements to the provided template imrpove a favorites system used to track message updates from favorited users and conversations, persistant storage of conversations and messages, and conversation access control that allows users to have private conversations.

Worked on in collaboration with Erick Velez and Julia Chin.

breakpro (Google Computer Science Summer Institute 2016)

A web application designed to help people take more effective breaks.

It allows you to set timers for you breaks to prevent yourself from procrastinating. Its features include the ability to add friends and share your break logs with one another to see who can stick to their break timers the most frequently.

Created in collaboration with Giovy Webb and Eve Coulibaly.
